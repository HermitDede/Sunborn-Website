Character

At the beginning, there will be only one character which is the main protagonist. Rest will be random characters created by some variables. Because of working solo on this project, add background stories to all citizens in the town seemed secondary. If the game development finishes, there won’t be any reason to add more characters.

  • Commander < You >

    • Protagonist of the game. < You > are a commander assigned to investigate a mysterious event occurred in a distant island. However, because of the situation, < You > find yourself as governor of a town called Sunborn and instead of investigating the event by yourself, < You > start training and sending adventurers.

  • < Adventurer >

    • Adventurers are randomized characters which will be played by the player if they choose them. Their random attributes will be background story, name, age, gender, race, class, status points, level and traits. Background story, name, age, gender will have no effect on gameplay. They are only for role playing elements. However, race, class, status points, level and traits will directly affect the characters’ strength and behavior.


Currently, the alpha version of the game allows to choose race and class for the character. The options are few, but mechanics will be similar to the release version of the game. You can check the character's race and class system of game's alpha version.

  • Race
  • Class

In HoDT: Sunborn, players can choose a race for their characters which gives some special benefits depending on the race chosen. Each race has advantages on various conditions and sometimes has disadvantages. Players should decide carefully by also taking consider of their chosen class for the character. Currently, the game includes 4 races for characters.

Character Sprites

Character class defines character's special abilities and its aptitudes for certain weapons or stats. It is not a must to choose a race or improve stats depending on the character's class. Players are free to make any kind of combination with status points, races and classes. Each class has 5 unique skills which has different kind of behaviours and usage ways. Some skills might add permanent effects while some skills casted upon enemies to deal damage. Players may see the explanation of each skill by hovering the mouse on to the skill. Current version only includes one class which is warrior, however the release version is planned to include at least 4 classes. Below, you can see the class selection case of the alpha version.
